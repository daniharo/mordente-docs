# Crea un evento

Desde el menú de la agrupación, podrás usar el botón `Ver eventos` para ver los eventos que se han creado. Si no has creado ninguno, verás un mensaje indicándolo.

Si quieres crear un evento, puedes usar el botón `Crear evento` en la lista de eventos. Te pedirá que introduzcas el nombre del evento y la fecha en la que se celebrará. Si quieres, puedes añadir una descripción.

## Publica el evento

Al terminar de crear el evento, se te mostrará un mensaje con el evento creado. Si quieres publicarlo, puedes usar el botón `Publicar`. Esto enviará un mensaje a todos los miembros de la agrupación con la información del evento.

Si eliges no publicarlo, se guardará como borrador para que puedas publicarlo más adelante.

## Asigna miembros

El bot te preguntará si deseas asignar todos los miembros al evento. En caso afirmativo, se asignarán automáticamente. Si no, podrás asignarlos manualmente más adelante.

En cualquier caso, los usuarios asignados recibirán una notificación con la información del evento.
