# Introducción

Descubramos **Mordente en menos de 5 minutos**.

## Empezando

Comienza **creando una agrupación**.

### Lo que necesitarás

Solo necesitas tener instalado [Telegram](https://telegram.org/dl) en tu dispositivo.

:::tip

Si estás en un ordenador, puedes usar [Telegram Web](https://web.telegram.org/) para no tener que instalar nada.

:::

## Iniciando el bot

Abre el bot de Telegram buscando `@mordente_bot` dentro de Telegram o haciendo clic aquí:

<div style={{ display: "flex" }}>
  <Link
    className="button button--primary button--lg"
    style={{ margin: "0 auto 1rem" }}
    href={BOT_URL}
  >
    Abrir el bot
  </Link>
</div>

El bot te dará la bienvenida cuando pulses el botón de `Iniciar`.

## Creando una agrupación

Al iniciar el bot, el bot te preguntará si quieres crear una agrupación. Si pulsas `Crear`, el bot te preguntará el nombre y la agrupación ya estará creada.
